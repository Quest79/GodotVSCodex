# Item Tooltip Icon Manifest

These transparent PNG assets were derived from the supplied `NEON HERESY` tooltip reference for the future modular item-card renderer.

| Asset | Intended use |
| --- | --- |
| `stat_physical_damage.png` | Physical damage stat row |
| `stat_critical_strike_chance.png` | Critical strike chance stat row |
| `stat_critical_strike_multiplier.png` | Critical strike multiplier stat row |
| `stat_attack_speed.png` | Attack speed stat row |
| `stat_armor_penetration.png` | Armor penetration stat row |
| `stat_life_on_kill.png` | Life-on-kill stat row |
| `decor_top_occult_crest.png` | Decorative crest centered above the item name |
| `decor_bottom_skull_medallion.png` | Rarity/binding medallion centered on the tooltip footer |

The six `stat_` assets are semantic stat icons. The two `decor_` assets belong to the card frame and should not be used as item modifiers.
